# Plugin FC sample

![](https://dev.lutece.paris.fr/plugins/plugin-fcsample/images/fc.png)

Ce plug-in est une version LUTECE de l'exemple de démonstration d'un fournisseur de services FranceConnect fourni à l'origine pour nodeJS.

La démo originale est disponible sur https://partenaires.franceconnect.gouv.fr/code-fs

Pour construire une webapp contenant le plugin, exécutez:

```

mvn lutece:exploded
                
```

Configurez les points de terminaison (endpoints) et les jetons dans le `franceconnect_context.xml` 

Pour plus de détails, voir la [documentation du plugin FranceConnect](https://github.com/lutece-secteur-public/franceconnect-plugin-franceconnect) 

Lancez l'application Web, activez le plug-in si ce n'est pas le cas et accédez à l'URL suivante:

```

http://myhost/jsp/site/Portal.jsp?page=fcsample
                
```
